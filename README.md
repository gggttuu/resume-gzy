# 第一次课后作业 index1.html

## 项目简介

本项目为《字节跳动技术训练营-前端》课程作业，实现了一个个人简历单页网站。  
主要内容为个人基本信息、教育背景、科研经历、实习项目以及获奖情况。

页面示例基于本人简历内容进行排版与展示。

---

## 技术栈与实现要求对应

1. **HTML 单页面 & 语义化标签**
   - 使用 `header / section / article / footer` 等语义化标签组织页面结构。
   - 整个项目只有一个 `index.html` 文件，构成完整的 DOM 骨架。

2. **CSS 布局与样式**
   - 使用 **Flex** 和 **CSS Grid** 完成页面布局：
     - `header` 使用 Flex 实现头像与个人信息的横向排列；
     - 教育背景、科研经历、实习经历等模块使用 Grid 实现多列表格化排版。
   - 通过盒模型、边框、间距等样式实现类似纸质简历的排版效果。

3. **JavaScript 交互功能**
   - 使用原生 JavaScript 实现“项目经历展开 / 折叠”功能：
     - 点击科研经历中的项目卡片标题，可以展开或收起详细内容；
     - 同时仅允许最多一个项目处于展开状态，提升可读性。
